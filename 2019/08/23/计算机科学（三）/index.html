#常量与变量

常量

常量是指在程序运行中，其数值不能被改变的量。
常量又可分为直接常量和符号常量。
直接常量：如：0，17，-99，3.14，”china”
符号常量：有时为了使用方便，可用一个符号名来代表一个常量。
定义形式：

#define 标识符 常量数据

优点：见名知义，一改全改
注意：一般符号常量名用大写，变量名用小写，符号常量不能被赋值和指定类型。

举个例子：

#define PI 3.14159
#include <stdio.h
void main()
{ float r,l;
r=3;
l=2PIr;
printf(“length=%f”,l);
}

变量

变量是指以标识符为名字，其值可以改变的量。
变量代表计算机内存中的某一存储空间，该存储空间中存放的数据就是变量的值。
变量的两个要素：
（1）变量名。每个变量都必须有一个名字–变量名，变量命名遵循标识符命名规则。
（2）变量值。在程序运行过程中，变量值存储在内存中。在程序中，通过变量名来引用变量的值。

在同一程序块中，变量不能被重复定义。
“先定义，后使用”。

###标识符命名规则：
（1）由字母，数字和下划线3种字符组成，且第一个字符必须为字母或下划线。
（2）关键字不能用作变量名。
（3）见名知意。

变量的定义和初始化

在C语言中要求对所有用到的变量，必须先定义、后使用；且称在定义变量的同时进行赋初值的操作为变量初始化。
(1)定义形式： 类型标识符 变量名1 [,变量名2，变量名3…];
(2)变量的初始化:在说明变量的同时给变量一个初始值。

变量作用域

变量的作用域分为：局部变量和全局变量。
局部变量：1.在函数内定义；只在该函数中才能使用。或在复合语句中定义和使用。
2.定义为函数的参数；只在该函数中才能使用。
全局变量（外部变量或全程变量）：在函数外定义；从它被定义的位置起，在之后的所有函数中有效。

#运算符和表达式

运算符：就是表示某种运算功能的符号。
按操作功能大致分为：算术运算符、关系运算符、逻辑运算符、按位运算符以及其他运算符如赋值运算符、条件运算符、逗号运算符等。
表达式：是由操作数和运算符组成的序列。

优先级顺序：

注意： /、%
（1）关于除法运算：两个整数相除，其商为整数，小数部分被舍弃（整除）。
（2）关于求余数运算：要求两侧的操作数均为整数型数据，否则出错。

优先级基本和数学上是差不多的。
fabs(-5)=5 (取正）

自增自减运算符

++i，–i先增减，后运算；i++，i–先运算，后增减。

i=2;j=-i++; ‘++’和’-‘是同优先级，所以i的值为3，j的值为-2。

说明：
（1）自增自减运算，常用在循环语句中，使循环变量加或减1；以及指针变量中，使指针指向下或上一个地址。
（2）不能用于常量和表达式。 （如：8++，–（x+y）都是非法的。）

关系运算符和关系表达式

（1）6种关系运算符：<,>,<=,>=,==,!=
（2）关系表达式：用关系运算符将表达式连接起来的式子
关系表达式的值不是1（TRUE)就是0(FALSE)。

逻辑运算符和逻辑表达式
(1) !(非）,&&（与）,||（或）。
优先级自左至右。
参与逻辑运算的操作叫逻辑量。
逻辑量可以是整型、实型、字符型和指针类型。

偷懒的&&和||
在逻辑表达式中，并不是所有逻辑运算符都被执行，知识在必须执行下一个逻辑运算符才能求出表达式的解时，才执行该运算。

#####逻辑表达式一般用于控制语句中的条件。
（1）数学上的区间：x <= x <= 8
代码化： x>2 && x<8
（2）year是闰年：能被4整除但不能被100整除或能被400整除
代码化：（year % 4 ==0 && year % 100 != 0）|| year % 400 == 0

“=”就是赋值运算符，它的作用是讲右边的一个表达式的值赋给左边的一个变量。
变量=赋值表达式
优先级：！> 算术 > 关系 > &&和|| >赋值

如果表达式值的类型，与被赋值变量的类型不一致，但是都是数值型或字符型时，系统自动地将表达式的值转换成被赋值变量的数据类型，然后再赋值给变量。

复合赋值运算符

复合赋值运算符是由赋值运算符之前再加一个双目运算符构成的。
复合赋值运算的一般格式为：表达式1 op=表达式2
它等价于： 表达式1 = 表达式2 op（表达式2）

其他运算符

#####（1)条件运算符
？和：，唯一的三元运算，需要三个操作数。
形式为：<表达式1>？<表达式2>：<表达式3>

（2）逗号运算符
形式：表达式1，表达式2，…,表达式n

求解过程：自左至右，以此计算各表达式的值，最后一个表达式的值即为整个逗号表达式的值。
例： “(a=4*5,a*4),a+5”的值=25：先求解a=4*5,得a=20;再求a*4=80;最后求解a+5=25,所以逗号表达式得值为25。

注意：C语言常用逗号作为分隔符。（注意区分哦！）

#####（3）求字节运算符sizeof
作用：求得变量或某种数据类型所需的字节数。

自动转换

C语言允许整型，实型，字符型数据进行混合运算。在进行运算时，不同类型得数据要先转换成同一类型，然后进行运算。
转换方式：自动转换、赋值转换和强制转换

赋值转换

在赋值时将赋值符右边值得类型转换成与其左边变量类型一致的类型。
（1）实型->整型（或字符型）：去掉小数部分；
（2）整型（或字符型）->实型：补足有效位
（3）赋值号左右为char，int，short，long，unsigned型数据的，形如b=a:
a位数等于b位数：原样照赋；
a位数大于b位数（放不下）：截断高a-b位，送低b位；
a位数小于b位数（放不满）：
①a为有符号数：不满的高位补符号位知道充满；
②a为无符号数：不满的高位补0直到充满。

强制类型转换

形式：（类型名）表达式 如(int)(a+b)、（int)x

注意：强制类型转换时，原来变量的类型不会改变。即上面的变量x虽然被临时强转为int来使用，但x的类型还是原来的类型。
